## 🌙 What is Noor?

</div>

**Noor** (meaning *Light* in Arabic) is a full-featured, production-grade Islamic companion app engineered from the ground up with Flutter. It is the only app you need for your complete Islamic lifestyle — from reading the Holy Quran with Tajweed coloring, exploring thousands of verified Hadiths, finding the perfect Dua for every moment, tracking your prayers, to locating the Qibla wherever you are on Earth.

Built with a passion for both **deen** and **design**, Noor combines a premium dark-first aesthetic with deep functionality powered by a clean BLoC architecture, offline-first data strategy, and multi-language support for Arabic, Urdu & English.

**Pattern:** Feature-first folder structure with BLoC state management

```
User Action
    │
    ▼
 Widget/UI
    │  dispatches Event
    ▼
  BLoC
    │  calls
    ▼
 Service / Repository
    │  fetches from
    ▼
 API / SQLite / Hive / JSON
    │  emits State
    ▼
  BLoC  ──▶  Widget rebuilds
```

## 🌍 Localization

Three languages, one seamless experience:

| Language | Script | Reading Direction | Font |
|---|---|---|---|
| **Arabic** (العربية) | Naskh / Uthmani | Right-to-Left ← | UthmanicHafs, DigitalKhatt IndoPak |
| **Urdu** (اردو) | Nastaleeq | Right-to-Left ← | Jameel Noori Nastaleeq Kasheeda |
| **English** | Latin | Left-to-Right → | Google Fonts (system default) |

Custom embedded fonts included in the app:
- `UthmanicHafs_V22.ttf` — Standard Quranic typography (15-line layout)
- `DigitalKhattIndoPak.otf` — Tajweed-rule-colored Arabic rendering
- `Jameel Noori Nastaleeq Kasheeda.ttf` — Premium Urdu Nastaleeq
- `A Thuluth Regular.ttf` — Classical Arabic display calligraphy
- `alqlm-asma-ul-husna.ttf` — Specialized Asma ul Husna calligraphy
- `word-by-word and translation.ttf` — Word-by-word Quran display
- `surah-name-v2.ttf` / `surah-name-v4.ttf` — Surah name decorative glyphs
- `icomoon.ttf` + `homepage.ttf` — Custom Islamic icon sets

## 🎨 Design Philosophy

> **"Less, but better."** — Dieter Rams

Noor is built with five core design principles:

1. **Dark-First** — Optimized for night reading and comfortable long sessions
2. **Typography-Led** — Premium Arabic calligraphy as the visual centerpiece
3. **Glassmorphism** — Frosted-glass surfaces with depth and luminosity
4. **Motion-Intentional** — Micro-animations that guide, not distract
5. **RTL-Native** — Every layout, every widget built with bi-directionality in mind

Premium UI elements include:
- Custom **Liquid Glass buttons** with animated shimmer
- **Shimmer skeleton** loading screens for instant perceived performance
- **Sliver app bars** with parallax header images
- **Draggable bottom sheets** for contextual actions
- **Haptic feedback** for key interactions

## 🌐 API Sources

| API | Base URL | Used For |
|---|---|---|
| **AlQuran.cloud** | `https://api.alquran.cloud/v1/` | Quran text, translations, editions |
| **Quran.com API v4** | `https://api.quran.com/api/v4/` | Tajweed data, word-by-word |

## 🚀 Getting Started

### Prerequisites

```
Flutter SDK  ^3.0.0
Dart SDK     ^3.0.0
Android SDK  API 21+ (Android 5.0 Lollipop and above)
Xcode        14+ (for iOS builds)
```

### Clone & Run

```bash
# Clone the repository
git clone https://github.com/your-username/quran_app.git
cd quran_app

# Install all dependencies
flutter pub get

# Run in debug mode
flutter run

# Run on a specific device
flutter run -d <device-id>
```

### Build for Production

```bash
# Android APK (sideload)
flutter build apk --release --split-per-abi

# Android App Bundle (Google Play)
flutter build appbundle --release

# iOS Archive (App Store)
flutter build ios --release
```

## 🔒 Privacy & Permissions

Noor respects your privacy. Here is every permission it requests and exactly why:

| Permission | Platform | Reason |
|---|---|---|
| `ACCESS_FINE_LOCATION` | Android / iOS | Calculate accurate Qibla direction |
| `INTERNET` | Android | Hadith API calls + audio streaming |
| `RECEIVE_BOOT_COMPLETED` | Android | Reschedule prayer notifications after reboot |
| `VIBRATE` | Android | Tasbeeh counter haptic feedback |
| `WAKE_LOCK` | Android | Keep screen on while reading Quran |
| `NSLocationWhenInUseUsageDescription` | iOS | Qibla compass direction |

**No analytics. No tracking. No ads. No data collection.**
Everything stays on your device.
